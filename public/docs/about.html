About table-sort-js

Objectives of table-sort-js:


  • Be easy to use. (use class "table-sort" on table tags).
  • No dependencies; keeps the size of the file down. For example jquery is around 80KB.
  • Be versatile; sort dates, numbers(in natural order), alpha and alphanumeric, etc.
  • Not interfere with the css of tables that should be left to the user, or via librarys such Bootstap CSS.
